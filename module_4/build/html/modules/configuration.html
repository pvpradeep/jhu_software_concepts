Configuration

Configuration Settings for Graduate Application Data Analysis System

This module contains all configuration parameters for the application, including database connection settings and file path configurations.

Database Configuration

src.config.DB_HOST: str = 'localhost'

Hostname where the PostgreSQL database is running

src.config.DB_PORT: int = 5433

Port number for PostgreSQL database connection

src.config.DB_NAME: str = 'jhudb_gradrec'

Name of the PostgreSQL database

src.config.DB_USER: str = 'jhuuser'

Username for database authentication

src.config.DSN: str

Complete database connection string formatted for psycopg

File Path Configuration

src.config.JSON_DATA_FILEPATH: str

Absolute path to the directory containing JSON data files

src.config.JSON_DATA_FILENAME: str = 'applicant_data'

Base filename for JSON data files. Files are named as {JSON_DATA_FILENAME}_{index}.json
